### new Renderer(config)
A class to render a scene


| Param | Type | Description |
| --- | --- | --- |
| config | <code>Object</code> | configuration object |
| config.scene | <code>Scene</code> | Scene to render |
| config.context | <code>Context</code> | WebGL context to render to |
